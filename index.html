Проект

Этот сайт разработан в качестве итогового проекта учеником Приходько Глебом из группы О722Б. Целью данного сайта является показать на примере разницу между POST и GET запросом для студентов.

Стек технологий

Этот сайт использует в себе не только html, css, js технологии, а также более специфичные и узконаправленные технологии такие как:

  • Bootstrap 5: Это популярный фреймворк для создания адаптивных и красивых сайтов. Сайт использует Bootstrap для создания сетки, формы, списка и кнопки. Сайт также подключает Bootstrap через CDN (Content Delivery Network), чтобы ускорить загрузку сайта.
  • jQuery: Это библиотека JavaScript, которая упрощает работу с DOM (Document Object Model), событиями, анимацией и AJAX (Asynchronous JavaScript and XML). Сайт использует jQuery для создания и добавления элементов на сайт, обработки событий формы и кнопки, отправки и получения данных от API с помощью AJAX и показа и скрытия иконки загрузки. Сайт также подключает jQuery через CDN, чтобы ускорить загрузку сайта.
  • JSON Placeholder: Это бесплатный API, который предоставляет фиктивные данные в формате JSON. Сайт использует JSON Placeholder для получения и создания постов. Сайт отправляет GET запрос к API, чтобы получить список постов, и POST запрос к API, чтобы создать новый пост.

Для взаимодействия с JSON Placeholder API используются GET и POST запросы.
